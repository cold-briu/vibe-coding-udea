# Mapita Explorador
Una herramienta que permite tomar datos urbanos que existentes y organizarlos de forma clara para entender mejor cada barrio/comuna. A partir de variables como condiciones demográficas, acceso a servicios y percepciones ciudadanas, se ordena la información y genera un perfil sencillo de cada territorio. 
La idea es facilitar la lectura de datos que están dispersos y no relacionados para convertirlos en descripciones claras.
